MatchMaker Application

Why do you need the Match Maker application?

Selecting the right tool to support collaborative work is one of the main problems teams face. There is a wide array of collaboration tools to choose from, thus selecting an appropriate tool can be a overwhelming and a significant challenge.

We know that the success of a project and even the efficiency of the team is based on the tool selected for collaboration. We are here to make such task easier.

What is Match Maker application?

MatchMaker Application is a recommendation system that helps users selecting appropriate collaborative tools based on the articulation of their work through the MoCA framework (you don't have to worry about this!).

Our application, through the search app feature will allow any user to describe the type of collaborative work to be done. As a result you get a list of appropriate tools for the work.

Also, if you want to contribute to the MatchMaker Application database, you can register collaborative computer-based tools using the register apps feature.
